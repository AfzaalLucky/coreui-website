Migrating to v3.0.0

CoreUI 3.0.0 is a major rewrite of the entire project. The most notable changes are summarized below, followed by more specific changes to relevant components.

On this page

    • Summary
      • Global changes
    • By component
      • Header
      • Sidebar
      • Brand Card and Brand Buttons
    • By CSS classes and attributes

Summary

Here are the big ticket items you’ll want to be aware of when moving from v2.x.x to v3.0.0.

Global changes

  • By default, all classes and attributes in CoreUI start with the c- prefix. This avoids name collisions when introducing CoreUI to existing projects or when combining it with other frameworks. CoreUI allows to change that prefix. This even allows to use multiple versions of CoreUI alongside each other.
  • CoreUI’s and Bootstrap’s styles and scripts were combined together. From this version, you don’t have to include Bootstrap’s files.
