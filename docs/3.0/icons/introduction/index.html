Installation

CDN

NPM

npm install @coreui/icons --save

Or, you can also clone or download this repository as zip.

Basic Use

You can place CoreUI Icons just about anywhere using a CSS style prefix and the iconâ€™s name. CoreUI Icons are designed to be used with inline elements ex. <i> or <span>.
