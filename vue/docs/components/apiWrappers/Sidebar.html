fixed true Boolean
Sets sidebar position to fixed. Prop is disabled if 'overlaid' prop is true.
unfoldable false Boolean
[PRO] Sets sidebar to unfoldable mode. Use instead of minimized prop.
overlaid false Boolean
[PRO] Sets sidebar position to overlaid. Prop have precedens over fixed prop.
breakpoint 'lg' String
Breakpoint of the transition between mobile and non-mobile mode. Valid values: 'sm', 'md', 'lg', 'xl'.
minimize false Boolean
Sets minimized state. You can track the curent state by .sync modifier.
show 'responsive' Boolean/ String
Sets state of the sidebar, available values:
  • true - sidebar is opened on all widths
  • false - sidebar is closed on all widths
  • responsive - sidebar is closed on widths below set breakpoint, and open above breakpoint width.
    Option is not available when 'overlaid' prop is set to true
You can track the curent state by .sync modifier.
hideOnMobileClick true Boolean
Enables mobile sidebar closing trigger. Trigger works on clicking on CSidebarNavItem link, or any other 'a' tag element with class 'c-sidebar-nav-link' inside sidebar wrapper.
aside false Boolean
Sets sidebar to placement to right side.
colorScheme 'dark' String
Sets sidebar color scheme.
dropdownMode 'openActive' String
Define CSidebarNavDropown components behavior on route change. CSidebarNavDropdown route state is active, if its 'route' prop is included in current vue-router route. Valid values:
  • openActive - opens dropdown when route state is active
  • close - closes every dropdown on route change
  • closeInactive - closes every inactive dropdown on route change, opens dropdown when route state is active
  • noAction - no automatic dropdowns toggling
