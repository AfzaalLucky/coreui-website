col, xs, sm, md, lg, xl false Boolean/String/Number/Object
Bootstrap grid classes will be generated based on values passed to these props.
Props can be used in following modes:

  • Boolean - column class with no specific value. Example:

    Input: 
    <CCol md>[...]</CCol>
    Output: 
    <div class="col-md">[...]</div>
  • String/Number - column class with specified columns number. Example:

    Input: 
    <CCol lg="6" sm="12">smh</CCol>
    Output: 
    <div class="col-lg-6 col-sm-12">[...]</div>
  • Object - Use for advanced column definition. Available keys: size, order, offset. Example:

    Input: 
    <CCol :md="{ size: '6', order:'2' }" :lg="{ offset:'3' }">[...]</CCol>
    Output: 
    <div class="col-md-6 order-md-2 offset-lg-3">[...]</div>
