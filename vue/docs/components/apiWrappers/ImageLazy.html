loadInitially false Boolean
Load image initially, not waiting for scroll.
loadOffset 500 Number
Distance range in pixels between scroll position and image. When distance is smaller than given value image will start loading. Works for scrolling down, as well as from bottom up. Value can be negative, in case in which value will be too small (image never renders), value will be reduced dynamically to minimal but stable value.
noFade false Boolean
Disable animation of an image.
fadeOffset -100 Number
Distance range in pixels between scroll position and image. When distance is smaller than given value image will start animating. Works for scrolling down, as well as from bottom up. Value can be negative, in case in which value will be too small (image never renders), value will be reduced dynamically to minimal but stable value.
fadeTime 1500 Number
Animation time in miliseconds.
