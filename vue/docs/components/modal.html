    # CMedia API

    In all slots there is available scoped method 'hide()' which emmits two events: 'update:visibility' and 'accepted'.

      'hide()' method is also executed when clicking on one of following triggers:
    • Close button in the header
    • 'OK' and 'Cancel' buttons in the footer
    • Backdrop (when noCloseOnBackdrop prop is not true)
