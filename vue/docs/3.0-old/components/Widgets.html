        # Widget with progress bar

        Widget which uses CProgress component.

        # CWidgetProgress API


        # Slot Example

        If you want to use custom progress bar you can use CProgress component in default slot. In that case, value and variant properties would be ignored by CProgress. You can also place in slot any other content.

        Custom CProgress

          # Widget Icon

          This widget can be displayed with or without a link. If you want to display footer with link just set the link property with the address as value.

          # CWidgetIcon API


          # CWidgetIcon example:

          <CWidget02
  header="$1.999,50"
  iconClasses="fa fa-cogs bg-primary"
  text="Income"
  link="https://coreui.io/vue/"
  linkText="Click here!"
/>

          # Widget social

          This widget style depends on variant property which defines which background color and font awesome icon will be used (i.e facebook, twitter, linkedin).

          # CWidgetSocial API


          # Basic example

          <CWidgetSocial
  variant="facebook"
  rightHeader="89k"
  rightFooter="friends"
  leftHeader="459"
  leftFooter="feeds"
/>

          Example with customized icon and chart.

          # Custom example

          <CWidgetSocial
  variant="warning"
  iconClasses="fa fa-battery-1"
  rightHeader="0:57"
  rightFooter="time"
  leftHeader="17"
  leftFooter="percentage"
>
  <CSimpleLineChart
    :data="[65, 59, 84, 84, 51, 55, 40]"
    label="Friends"
    :labels="['January', 'February', 'March', 'April', 'May', 'June', 'July']"
    style="height:100px"
    color="rgba(255,255,255,.1)"
  />
</CWidgetSocial>

          # Widget with progress bar and icon

          This widget style can be configured by changing variant property. If you want to change the color of widget instead of changing progress bar color set the inverse property to true.

          # CWidgetProgressIcon API


          # Basic Example

          <CWidgetProgressIcon
  iconClasses="icon-basket-loaded"
  header="1238"
  text="Products sold"
  variant="background-warning"
/>

          # Slot Example

          If you want to use custom progress bar you can use CProgress component in default slot. In that case, value and variant properties would be ignored by CProgress. You can also place in slot any other content.

            # Widget dropdown

            This widget style is defined by variant. Inside component there are slots for dropdown and chart.

            # CWidgetDropdown API


            # Example with CDropdown and chart

            <CWidgetDropdown
  variant="info"
  header="9.823"
  text="Members online"
>
  <template #dropdown>
    <CDropdown
      class="float-right"
      variant="transparent p-0"
      right
      buttonHtml="<i class='icon-settings'></i>"
    >
      <CDropdownItem>Action</CDropdownItem>
      <CDropdownItem>Another action</CDropdownItem>
    </CDropdown>
  </template>
  <template #default>
    <CSimplePointedChart
      class="px-3"
      :data="[65, 59, 84, 84, 51, 55, 40]"
      :labels="['January', 'February', 'March', 'April', 'May', 'June', 'July']"
      style="height:70px"
      label="Members"
      pointHoverColor="primary"
    />
  </template>
</CWidgetDropdown>

            # Widget small text

            This widget style is defined by variant. Inside component there is default slot for additional data (i.e. charts).

            # CWidgetSmallText API


            # Example

            <CWidget07
  variant="warning"
  rightHeader="SALE"
  rightFooter="Today 6:43 AM"
  leftHeader="$1.890,65"
  leftFooter="+432,50 (15,78%)"
>
  [Charts]
</CWidget07>
