# Introduction

A @coreui/vue v3-alpha components library project

# Over 60 bootstrap based vue.js components!

Check out template build on top of this library: CoreUI Open Source Bootstrap Admin Template

# Installation

npm install @coreui/vue

@coreui/vue can be used as a module in both CommonJS and ES modular environments.

When in non-modular environment, @coreui/vue will register all the components to vue by itself.

# ES6

//
// You can register a component manually
//
import { CSwitch } from '@coreui/vue';

export default {
  ...
  components: {
    CSwitch
  },
  ...
};

//
// or register the whole module with vue
//
import CoreuiVue from '@coreui/vue';

// Install this library
Vue.use(CoreuiVue);
